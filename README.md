# RingWars 3D

Fresh 3D reboot inspired by RINGwars2.0.

## Stack
- Three.js
- Vite
- TypeScript

## Quick start
```bash
npm install
npm run dev
```

## Goals (Phase 1)
- Third-person arena camera
- Player movement + dash
- Basic enemy AI (seek + orbit)
- Projectile system
- Health + score HUD
- Polished lighting + post effects (bloom + vignette)
